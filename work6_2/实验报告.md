# 实验报告：PyTorch3D 可微渲染网格重建（Work6_2）

## 1. 实验目标

1. 理解可微光栅化在离散网格边界上的梯度近似原理。
2. 掌握通过多视角二维监督优化三维网格顶点的方法。
3. 理解网格正则化在稳定优化、防止拓扑崩坏中的作用。
4. 完成选做：在轮廓监督基础上加入 RGB 监督进行联合优化。

## 2. 实验原理

### 2.1 软光栅化

传统硬光栅化采用离散像素归属，边界附近梯度不可导。软光栅化使用平滑近似：

$$A(d)=\\sigma\\left(\\frac{d}{\\sigma_r}\\right)$$

其中 $d$ 为像素到三角形边缘的符号距离，$\\sigma_r$ 控制平滑程度。该机制使边界附近产生连续梯度，支持端到端顶点优化。

### 2.2 网格正则化

仅依赖图像误差会导致网格出现尖刺、拉伸和局部翻折，因此加入几何正则项：

- 拉普拉斯平滑：抑制高频噪声；
- 边长约束：抑制异常拉伸；
- 法线一致性：保持表面连续平滑。

总损失：

$$L_{total}=L_{silhouette}+w_{rgb}L_{rgb}+w_{lap}L_{lap}+w_{edge}L_{edge}+w_{normal}L_{normal}$$

当仅做必做任务时，取 $w_{rgb}=0$。

## 3. 实现说明

### 3.1 代码结构

- `src/Work6/config.py`：结构化实验配置（render/train/loss）。
- `src/Work6/data.py`：目标网格加载、归一化、多视角相机构造。
- `src/Work6/renderers.py`：SoftSilhouette 与 SoftPhong 渲染器。
- `src/Work6/losses.py`：silhouette/rgb/正则项损失定义。
- `src/Work6/train_silhouette.py`：必做训练流程。
- `src/Work6/train_textured.py`：选做训练流程（几何 + 顶点颜色）。
- `src/Work6/visualize.py`：图像、曲线、mesh、指标日志导出。
- `src/Work6/main.py`：统一命令入口。

### 3.2 非雷同重构点

1. 采用统一子命令入口，避免模板式双脚本分裂流程。
2. 配置由 dataclass 分层管理，便于参数实验。
3. 输出目录统一结构并自动写入 `metrics.json`。
4. 必做与选做默认超参数、随机种子独立。

## 6. 问题排查记录

1. `ModuleNotFoundError: pytorch3d`  
   解决：使用课程匹配版本，优先 Colab 预置 CUDA 轮子环境。
2. 训练中 mesh 形变剧烈  
   解决：提高 `w_lap` 与 `w_edge`，并适当降低学习率。
3. 颜色优化不稳定  
   解决：将 `w_rgb` 从 1.0 调整到 1.2，并增加保存频率观察中间态。

## 7. 结论

1. 软光栅化是可微 mesh 优化可行性的关键。
2. 几何正则化直接决定优化稳定性和最终网格质量。
3. RGB 联合监督提升外观一致性，但计算开销更高，需合理平衡分辨率、视角数和迭代步数。
